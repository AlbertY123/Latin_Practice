# Latin Practice Website (v3)

## Included now
- Verb parsing (single mixed pool, no levels)
- Pronoun full-grid drill in order: nominative, accusative, genitive, dative, ablative
- Expanded pronoun sets (hic/ille/is/iste/qui/ipse)
- Noun declension full-grid drill in same case order
- Unseen generator with only 2 levels: Hard and Very Hard
- Dactylic hexameter scansion workbench (syllable quantity + 6-foot division via 5 breaks + caesura)
- Green + gold theme

## Run
Open `index.html` directly in browser.

Optional local server:
```bash
python -m http.server 8080
```
then open `http://127.0.0.1:8080`.
